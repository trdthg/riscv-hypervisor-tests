# RISC-V Hypervisor Tests

## About

This repository contains a collection of tests for the RISC-V Hypervisor Extension. Currently, the following repositories are included:

- [riscv-hyp-tests](https://github.com/trdthg/riscv-hyp-tests)
- [riscv-hs-tests](https://github.com/trdthg/riscv-hs-tests)
- [riscv-hext-asm-tests](https://github.com/trdthg/riscv-hext-asm-tests)

All these are forked with some bug fix or more test cases, and they are not mentained by the original authors anymore.

## How to use

1. prepare riscv-gnu-toolchain, and `RISCV` environment variable

    Note: riscv-gcc from ubuntu apt or riscv-gnu-toolchain

2. check the Makefile and Readme of each repository
